🎯 CSS 基础选择器

学习CSS中最常用的基础选择器：元素选择器、类选择器、ID选择器和属性选择器。

1. 元素选择器 (Element Selector)

所有的h3标题都会应用这个样式

这是一个段落，所有p元素都会有相同的行高和边距。

元素选择器直接使用HTML标签名称作为选择器。

CSS代码： 
/* 选择所有h3元素 */
h3 {
    color: #007bff;
    border-bottom: 2px solid #007bff;
    padding-bottom: 5px;
}

/* 选择所有p元素 */
p {
    line-height: 1.6;
    margin: 10px 0;
}
2. ID选择器 (ID Selector)

这个元素有唯一的ID，只有这一个元素会被选中。

这是一个带ID的标题

CSS代码： 
/* 选择ID为unique-element的元素 */
#unique-element {
    background-color: #fff3cd;
    border: 1px solid #ffeaa7;
    padding: 15px;
    border-radius: 4px;
}

/* 选择ID为header-title的元素 */
#header-title {
    color: #dc3545;
    font-size: 1.5em;
    text-align: center;
}
注意：ID在页面中必须是唯一的，每个ID只能使用一次。
3. 类选择器 (Class Selector)

这是一个高亮的内容块，使用了highlight类。

这是一个警告信息，使用了warning类。

这段文字应用了blue-text类，所以是蓝色的。

这段普通文字没有应用任何类。

这段文字也应用了blue-text类。

CSS代码： 
/* 选择所有class为highlight的元素 */
.highlight {
    background-color: #d4edda;
    border-left: 4px solid #28a745;
    padding: 10px;
    margin: 10px 0;
}

/* 选择所有class为warning的元素 */
.warning {
    background-color: #f8d7da;
    border: 1px solid #f5c6cb;
    color: #721c24;
    padding: 10px;
    border-radius: 4px;
}

/* 选择所有class为blue-text的元素 */
.blue-text {
    color: #007bff;
    font-weight: bold;
}

    border: 2px solid #007bff;

input[type="email"] {
    border: 2px solid #28a745;
    padding: 8px;
    border-radius: 4px;
}

/* 选择href属性以https开头的a元素 */
a[href^="https"] {
    color: #28a745;
    font-weight: bold;
}

/* 选择data-priority属性为high的元素 */
[data-priority="high"] {
    background-color: #f8d7da;
    border-left: 4px solid #dc3545;
    padding: 10px;
}
5. 通配符选择器 (Universal Selector)

在这个容器内

所有元素 都会有虚线边框
CSS代码： 
/* 选择.universal-demo内的所有元素 */
.universal-demo * {
    border: 1px dashed #6c757d;
    margin: 2px;
    padding: 5px;
}
注意：通配符选择器会选择所有元素，使用时要谨慎，可能影响性能。

7. 选择器优先级

这段文字展示了选择器优先级

这段文字使用了!important

优先级从高到低： 
/* 1. !important 声明（最高优先级）*/
.priority-demo {
    color: blue !important;
}

/* 2. ID选择器 */
#priority-test {
    color: red;
}

/* 3. 类选择器 */
.priority-class {
    color: green;
}

/* 4. 元素选择器（最低优先级）*/
p {
    color: black;
}
结果：第一段文字是红色（ID选择器优先），第二段是蓝色（!important优先级最高）
