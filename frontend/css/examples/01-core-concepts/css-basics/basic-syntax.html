📝 CSS 基础语法

学习CSS的基本语法规则，理解选择器、属性、值和注释的写法。

1. CSS 语法结构

基本语法规则：

选择器 {
    属性名: 属性值;
    属性名: 属性值;
    /* 注释内容 */
}

这个框应用了CSS样式

背景色、内边距、边框和圆角都是通过CSS设置的。

对应的CSS代码： 
.syntax-demo {
    color: #333;                    /* 文字颜色 */
    background-color: #f8f9fa;      /* 背景颜色 */
    padding: 20px;                  /* 内边距 */
    border: 1px solid #dee2e6;      /* 边框 */
    border-radius: 4px;             /* 圆角 */
    margin: 10px 0;                 /* 外边距 */
}
2. 选择器类型

HTML 结构：

<h2>标题元素</h2>
<p id="unique">ID选择器</p>
<p class="demo-box">类选择器</p>
<span>元素选择器</span>

CSS 选择器：

h2 { color: blue; }        /* 元素选择器 */
#unique { color: green; }   /* ID选择器 */
.demo-box { color: red; }   /* 类选择器 */
span { color: purple; }     /* 元素选择器 */

3. 属性和值的类型

这个盒子展示了不同类型的CSS值

宽度用像素(px)、高度用em、边距用rem、颜色用多种格式

CSS代码说明： 
.value-types {
    /* 长度单位 */
    width: 300px;          /* 绝对单位：像素 */
    height: 2em;           /* 相对单位：相对于字体大小 */
    margin: 1rem;          /* 相对单位：相对于根元素字体 */
    
    /* 颜色值的不同写法 */
    color: red;                    /* 颜色名称 */
    background: #ff0000;           /* 十六进制 */
    border-color: rgb(255, 0, 0);  /* RGB函数 */
    
    /* 百分比值 */
    padding: 5%;           /* 相对于父元素的百分比 */
    
    /* 关键字值 */
    text-align: center;    /* 预定义关键字 */
    display: block;        /* 布局关键字 */
}

5. 常见语法错误

注意CSS语法规则

每个属性后面必须有冒号，每个声明后面必须有分号

错误示例对比： 
/* ❌ 错误的语法 */
.error-example {
    color red;                    /* 缺少冒号 */
    background-color: blue        /* 缺少分号 */
    font-size: 16px;
}

/* ✅ 正确的语法 */
.correct-example {
    color: red;                   /* 有冒号 */
    background-color: blue;       /* 有分号 */
    font-size: 16px;             /* 完整的声明 */
}
